About CalendarGPT

CalendarGPT is a smart, web-based assistant that helps you manage your Google Calendar using natural language. Once connected to your Google account, you can view upcoming events, create and edit events using natural phrases like "Lunch with Dan tomorrow at 1pm", and delete entries you no longer need.

This project was built as part of a personal learning journey and is shared freely as a useful tool for the community. It uses Google Calendar API securely, and your data is never stored or shared beyond what's necessary to fulfill your requests in real time.
