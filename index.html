About CalendarGPT

CalendarGPT is a web-based assistant that helps users manage their Google Calendar using natural language chat. The application allows you to communicate with an AI assistant to perform common calendar tasks.

After connecting your Google account, CalendarGPT enables you to:

  • View upcoming events
  • Create new events with natural language (e.g., "Schedule lunch with Dan tomorrow at 1pm")
  • Edit existing events
  • Delete events you no longer need

This project was developed as part of a personal learning journey and is offered freely as a helpful tool for the community. The app uses the Google Calendar API to provide these features and requires your permission to access your calendar data securely.

User privacy is a core principle of this app: your data is never shared or stored beyond what is required to provide the requested features in real time.
